客房门口听叫床




昨晚小狼还是夜班，猥亵喝醉女那天是前天，说来这两天夜班到是收获不小，小狼现在都爱上了夜班。言归正传，小狼夜班带新来的同事，晚上11点多，在大堂无所事事，无聊至极。突然电话响了，我就让新来的同事去接，接完以后他脸都红了，说客人让买东西，我问他是不是避孕套，他红着脸说是。。。这同事一看就是刚从学校里出来的，买个避孕套都会脸红，哈哈。 然后我就告诉他买避孕套的地点，他就上楼找客人拿钱了。下楼的时候他在台子里找了个纸片写了一下房号，以免自己忘掉，小狼我当时就在旁边，侧目看了下，2113。 嘿嘿，2113的客人需要避孕套。。。。


　　说实话，小狼以前夜班也经常去买避孕套，买完给客人送去之后就下楼了，但是昨晚可能是太无聊了，小狼就萌生了一个大胆的想法，去客房门口听叫床。。。。左等右等，终于等到那个同事回来，看他送上楼，然后下来，一脸兴奋告诉我，挣了25快小费。 我打发他继续站在大堂值班，然后我就晃晃悠悠冒充巡视，一拐弯上了电梯，直奔2113。


　　刚开始到了2113，站在门口侧耳倾听，里面貌似传来女人呻吟，但声音很小，听不清，不知道是不是我心里的作用。听了一会，实在是听不出什么动静，再加上走廊里到处都有监视器，小狼怕站久了保安部会过来查，所以就下楼了。在楼上晃悠了大概20分钟，实在是不甘心，心里一直想着这个事情，小狼就决定再次返回2113。 这一次，站在门口，到时听到了里面那男的大口喘息，那女的到是没什么声音，听了有不到一分钟，那男的突然闷哼一声，小狼就估摸着这是完事了。。。。后来就听到了那男的去洗澡的声音，还听到男的对女的说 你也来洗洗吧， 那女的声音貌似很羞涩，怪不得刚刚听不到那女的声音。。。。感觉下面没什么内容了，小狼就悻悻的下楼了。。。


　　这次听床挺失败的，估计小狼第一次去的时候，里面正在前戏，也许是女的正在给男的吹，所以没什么动静，在小狼下楼的那段时间，才是他们高潮的部分，可惜小狼没有听到，只听到了一个结尾。。。。


　　这就是前两个夜班的经历，以后小狼再有什么经历会第一时间分享给大家，希望小狼也能早日升级~~~哈哈
