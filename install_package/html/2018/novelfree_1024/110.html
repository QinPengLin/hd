　　晚上我就和老婆商量了招个出纳的想法，老婆也同意了，只是她要招个能在市场里帮忙的，而我私下想出纳应该在库房。这样，小田就名正言顺的来我这上班了，尽管她成了老婆跟班。 

　　老婆经常和她要好的朋友出去逛街，这时我就呆市场里。看着可人的小田那别样的眼神，我几次按奈不住，就把看店的小玲支开，拉小田到里边换衣间强行泄火，至于平时没人的时候，摸摸奶子拧拧屁股什么的，更不在话下。
