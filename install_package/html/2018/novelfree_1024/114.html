我与按摩小姐




去年我刚毕业参加工作，在一家大公司做事，刚来公司也没有什么大事可以让我们亲来的做，所以晚上都有很多时间出去玩。

　　同事说应该不会很多，一次就180元吧，我想应该不多，又不是常去，就二个来到按摩中心。
